# PytorX_WARP
Pytorx_WARP for fault injection and charge trapping experiments in multi-bit FeFET crossbars. This tool is built on top of PytorX framework.

If you use any part of this work, please cite the following papers:

1) Original Pytorx paper: Zhezhi He, Jie Lin, Rickard Ewetz, Jiann-Shiun Yuan, and Deliang Fan. 2019. Noise Injection Adaption: End-to-End ReRAM Crossbar Non-ideal Effect Adaption for Neural Network Mapping. In Proceedings of the 56th Annual Design Automation Conference 2019 (DAC '19). Association for Computing Machinery, New York, NY, USA, Article 57, 1–6. https://doi.org/10.1145/3316781.3317870
